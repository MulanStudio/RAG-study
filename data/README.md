# 📂 数据目录

## 使用方法

**把组委会提供的所有数据文件放到这个文件夹！**

系统会自动识别并加载以下格式：

## 支持的文件格式

| 格式 | 说明 | 处理方式 |
|------|------|---------|
| `.pdf` | PDF 文档 | 自动提取表格 + 文本 |
| `.xlsx` | Excel 表格 | 每行转为独立记录 |
| `.csv` | CSV 数据 | 每行转为独立记录 |
| `.docx` | Word 文档 | 保留章节结构 |
| `.md` | Markdown | 按标题切分 |
| `.png` / `.jpg` | 图片 | VLM 生成描述 |

## 目录结构示例

```
data/
├── 财务报告.pdf
├── 季度数据.xlsx
├── 设备参数.csv
├── 服务合同.docx
├── 技术手册.md
├── 设备图片/
│   ├── rig_spec.png
│   └── safety_alert.jpg
└── 子文件夹/
    └── 更多文件...
```

## 注意事项

1. **支持子文件夹**：可以按类别组织文件
2. **文件名建议**：使用有意义的文件名，会被用作元数据
3. **编码**：确保文本文件使用 UTF-8 编码
4. **大文件**：PDF 超过 100 页可能需要较长处理时间

## 数据放好后

```bash
# 运行系统
python app.py

# 或启动 Web UI
streamlit run streamlit_app.py
```
